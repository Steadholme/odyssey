Components

Layout

页面骨架与排布原语：.ody-container 容器、.ody-stack / .ody-cluster / .ody-center / .ody-spread 排布语义、.ody-grid 与 .ody-flex-* 栅格、.ody-divider 分割线、.ody-aspect 比例盒，以及一套键控 --ody-sp-* 的间距工具类。全部零依赖，仅用语义 token，深色主题自动适配。

下方每个预览框都是真实渲染的 .ody- 布局，示例中的编号方块只是为了让栅格与排布可见。所有间距与半径都取自 --ody-sp-* / --ody-r-* token，切换深色即可对照。

Container

.ody-container 是居中定宽的页面包裹层，默认 max-width 1120px 并带左右 --ody-sp-6 内边距。修饰符 --narrow（760px，正文/表单）与 --wide（1320px，仪表盘）切换阅读宽度。

default · 1120px
--narrow · 760px
<div class="ody-container">…</div>
<div class="ody-container ody-container--narrow">…</div>
<div class="ody-container ody-container--wide">…</div>

Stack

.ody-stack 用 flex 纵向堆叠子元素，靠 gap 维持垂直节奏（默认 --ody-sp-4），无需给子项加 margin。--sm 收紧到 --ody-sp-2，--lg 放宽到 --ody-sp-6。

Row 1
Row 2
Row 3
<div class="ody-stack">         <!-- gap 16px -->
  <div class="ody-card">…</div>
  <div class="ody-card">…</div>
</div>
<div class="ody-stack ody-stack--sm">…</div>  <!-- gap 8px -->
<div class="ody-stack ody-stack--lg">…</div>  <!-- gap 24px -->

Cluster

.ody-cluster 是横向成组原语：flex-wrap + gap + 垂直居中对齐，专门放按钮、标签这类不定数量、需要换行的行内元素。--sm / --lg 调整间距。

draft v1.1.0 
<div class="ody-cluster">
  <button class="ody-btn ody-btn--primary">Save</button>
  <button class="ody-btn ody-btn--secondary">Cancel</button>
  <span class="ody-tag ody-tag--accent">draft</span>
</div>

Spread & Center

.ody-spread 把两侧内容推到行的两端（space-between），常用于卡片头/工具栏；.ody-center 则把单个子元素在两轴居中。行内两端对齐并需垂直居中时，也可用既有的 .ody-flex-between。

标题
居中内容 
<div class="ody-spread">
  <strong>标题</strong>
  <button class="ody-btn ody-btn--secondary ody-btn--sm">操作</button>
</div>

<div class="ody-center"><span>居中内容</span></div>

Grid

.ody-grid 是等宽栅格，配 --2 / --3 / --4 指定列数，gap 默认 --ody-sp-4。窄屏自动降列（4/3 列先降 2 列、再降 1 列）。需要按内容自适应列数时用 .ody-cols（auto-fit minmax）。

1
2
3
4
5
6
<div class="ody-grid ody-grid--3">
  <div class="ody-card">…</div>
  <div class="ody-card">…</div>
</div>

<!-- 内容自适应列数 -->
<div class="ody-cols">…</div>

Flex utilities

克制的一组 flex 工具：.ody-flex 起手，.ody-flex-center 垂直居中一行，.ody-flex-between 两端对齐并居中，.ody-flex-wrap 允许换行，.ody-flex-1 让某项吃掉剩余空间。间距用 .ody-gap-* 叠加。

fixed 
<div class="ody-flex ody-flex-center ody-gap-3">
  <span class="ody-tag">fixed</span>
  <div class="ody-flex-1">…</div>   <!-- 吃掉剩余空间 -->
  <button class="ody-btn ody-btn--sm">end</button>
</div>

Divider

.ody-divider 是取自 --ody-border 的水平分割线；.ody-divider--vert 是竖向分隔（需在 flex 行内、有高度约束时使用）；带文案时用 .ody-divider__label，两侧自动补线。

上段内容

下段内容

OR
左 右 
<hr class="ody-divider">
<div class="ody-divider__label">OR</div>
<span class="ody-divider--vert"></span>   <!-- flex 行内竖分隔 -->

Aspect box

.ody-aspect 是固定宽高比的盒子，子元素（图片/视频/占位）自动铺满并 object-fit:cover。比例修饰符 --16x9（默认）、--4x3、--1x1。

16 : 9
4 : 3
1 : 1
<div class="ody-aspect ody-aspect--16x9">
  <img src="cover.jpg" alt="">
</div>
<div class="ody-aspect ody-aspect--1x1">…</div>

Spacing utilities

间距工具类全部键控 --ody-sp-1..8（4/8/12/16/20/24/32px）。方向性 .ody-mt-* / .ody-mb-* 调外边距，全边 .ody-m-* / .ody-p-* 调外/内边距，.ody-gap-1..8 调 flex/grid 间隙。0 表示归零。

.ody-p-4 · 16px 内边距
.ody-mt-3 · 12px 上外边距 + .ody-p-2 
<div class="ody-p-4">…</div>          <!-- padding 16px -->
<div class="ody-mt-3">…</div>         <!-- margin-top 12px -->
<div class="ody-flex ody-gap-6">…</div><!-- gap 24px -->

<!-- 可用刻度：1=4 2=8 3=12 4=16 5=20 6=24 8=32 (px) -->

Display helpers

切换显示方式的小工具：.ody-hidden 彻底隐藏（display:none），.ody-block 转块级，.ody-inline-flex 行内 flex（自带垂直居中与 --ody-sp-2 间距，适合图标+文字徽标），.ody-w-full 撑满宽度。文本对齐用 .ody-text-center / .ody-text-right，单行省略用 .ody-truncate。

inline-flex 徽标
这是一段很长很长很长很长很长会被截断的文本
<span class="ody-inline-flex"><svg>…</svg>标签</span>
<div class="ody-truncate">很长的会被截断的文本…</div>
<div class="ody-hidden">不显示</div>
