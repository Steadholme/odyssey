Components

Overlays

悬浮在页面之上的轻量信息层：tooltip 提示气泡、popover 点击浮层与 hover card 富信息悬浮卡。前两者由 dist/odyssey.js 的 data-ody-* 属性零配置驱动，hover card 为纯 CSS 悬停展开，全部自动适配深浅主题。

下方每个预览框都是真实渲染的 .ody- 组件，右上角切换深色即可对照两套主题。把鼠标移上去、或点击触发器即可看到浮层实时出现。

Tooltip

给任意元素加 data-ody-tooltip="文本" 即可获得提示气泡：hover 悬停或键盘 focus 时出现，移开或按 Esc 关闭。浮层自动带 role="tooltip" 并通过 aria-describedby 关联触发器，无需任何 JS 初始化。

带提示的链接 
<button class="ody-btn ody-btn--secondary" data-ody-tooltip="保存当前更改">悬停查看提示</button>

<button class="ody-iconbtn ody-iconbtn--outline" data-ody-tooltip="设置" aria-label="设置">
  <svg viewBox="0 0 24 24">…</svg>
</button>

<a class="ody-link" href="#" data-ody-tooltip="也能挂在链接上">带提示的链接</a>

方向

用 data-ody-placement 指定气泡方向，取值 top（默认）/ bottom / left / right。JS 会实时测量并把浮层吸附到触发器对应边，同时做视口边界收拢，避免溢出屏幕。

<button class="ody-btn ody-btn--secondary" data-ody-tooltip="上方提示" data-ody-placement="top">Top</button>
<button class="ody-btn ody-btn--secondary" data-ody-tooltip="下方提示" data-ody-placement="bottom">Bottom</button>
<button class="ody-btn ody-btn--secondary" data-ody-tooltip="左侧提示" data-ody-placement="left">Left</button>
<button class="ody-btn ody-btn--secondary" data-ody-tooltip="右侧提示" data-ody-placement="right">Right</button>

Popover

在触发器上加 data-ody-popover="#id" 指向一个浮层容器：点击切换开关，再次点击、点击浮层外部或按 Esc 关闭。odyssey.js 自动补齐 aria-expanded / aria-controls 语义、给容器套上 .ody-popover 与 .is-open，并管理焦点返回。浮层里可放标题（.ody-popover__title）、说明文字与操作按钮。

分享此文档

任何持有链接的人都可以查看。切换为公开后会生成短链。

配额说明

当前计划每月包含 10,000 次请求，超出部分按量计费。可在账单页调整上限。

<div style="position:relative; display:inline-block">
  <button class="ody-btn ody-btn--secondary" data-ody-popover="#pop-share" aria-controls="pop-share">
    <svg viewBox="0 0 24 24">…</svg> 分享
  </button>

  <!-- 浮层容器：JS 会自动补上 .ody-popover / .is-open -->
  <div id="pop-share" style="top:calc(100% + 8px); left:0" hidden>
    <p class="ody-popover__title">分享此文档</p>
    <p>任何持有链接的人都可以查看。</p>
    <div class="ody-flex ody-gap-2">
      <button class="ody-btn ody-btn--primary ody-btn--sm">复制链接</button>
      <button class="ody-btn ody-btn--ghost ody-btn--sm">管理权限</button>
    </div>
  </div>
</div>

Hover card

.ody-hovercard 是一张富信息悬浮卡：包住一个触发器（如用户名、链接），悬停或键盘 :focus-within 时展开 .ody-hovercard__panel，展示头像 / 预览（__media）、标题（__title）、次要信息（__meta）与正文。它是纯 CSS 实现——无需 JS，面板与触发器之间有隐形桥接区，方便鼠标平滑移入。
