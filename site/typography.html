Components

Typography

排版基元：.ody-prose 富文本容器统一渲染整段 Markdown 式内容，独立引用块 .ody-blockquote、描述列表 .ody-dl、列表变体，以及一整套尺寸 / 字重 / 色调 / 截断的文本工具类。全部只引用语义化 --ody-* token，深浅两套主题自动生效。

下方每个预览框都是真实渲染的 .ody- 组件，右上角切换深色即可对照两套主题。行内代码 .ody-code、按键 .ody-kbd、链接 .ody-link 与等宽 .ody-mono 已内置于核心样式表，本页在需要处直接复用。

Prose

.ody-prose 是富文本容器：把一段由后端 / Markdown 渲染出来的裸 HTML 直接包进来，容器内的 h1–h4、p、ul/ol、a、strong/em、code/pre、blockquote、hr、table 都会获得基于 --ody-sp-* 的统一垂直节奏。样式严格作用域在 .ody-prose 内，不会污染宿主页面的裸标签。

安装与引入

Odyssey UI 是零依赖的单文件样式表，通过 一行 link 即可接入。它不劫持宿主页面的裸标签，仅对 .ody-* 生效。

  • 纯 CSS，无构建步骤
  • 深浅主题随系统或 data-theme 切换
    • 语义 token 自动继承
  • 可选的 odyssey.js 驱动交互
好品味在于消除特例，让边界情况自然消失。

Blockquote

独立引用块 .ody-blockquote 无需 .ody-prose 包裹即可单独使用，左侧以 --ody-accent 强调条标记。可选子元素 .ody-blockquote__cite 渲染出处，自动补上破折号前缀。

Description list

.ody-dl 是术语 / 描述对，默认两栏网格（术语列自适应、描述列占满）。子元素 .ody-dl__term 与 .ody-dl__desc。加 --stack 变体改为术语在上、描述在下的单栏堆叠，适合窄容器。

命名空间
所有类以 .ody- 前缀，token 以 --ody- 前缀。
主题
浅色默认，深色经 prefers-color-scheme 与 data-theme="dark" 双通道生效。
依赖
零外部请求，零运行时依赖。

  <dd class="ody-dl__desc">所有类以 .ody- 前缀……</dd>
  <dt class="ody-dl__term">主题</dt>
  <dd class="ody-dl__desc">浅色默认，深色双通道生效。</dd>
</dl>

List variants

在 .ody-list（重置了 margin/padding 与 list-style）之上叠加修饰符即可得到不同标记的列表：--plain 用低调圆点，--check 用 --ody-ok 勾选圆标，常用于特性清单。

  • 零依赖、零外部请求
  • 深浅双主题自动切换
  • Element-UI 级组件广度
  • 可 CDN 直接引用
  • 宿主页面裸标签不受影响
  • 语义 token 单点换肤

截断与多行省略

单行省略 .ody-truncate 已内置于核心样式表；本节新增多行钳制 .ody-line-clamp-2 / .ody-line-clamp-3，在第 2 / 3 行以省略号截断。.ody-nowrap 阻止换行但不省略。

单行省略：这是一段会在容器边缘被省略号截断的很长很长很长很长很长很长很长很长的文本。

两行钳制：这是一段较长的说明文字，超过两行的部分会被自动隐藏并在第二行末尾出现省略号，常用于卡片摘要、列表副标题等需要固定高度的场景，避免长文撑破布局。

三行钳制：这是一段更长的说明文字，超过三行的部分会被自动隐藏并在第三行末尾出现省略号。多行钳制依赖 -webkit-box 与 line-clamp，主流浏览器均已支持，适合摘要、评论预览、通知正文等固定高度容器。

.ody-nowrap 保持单行不换行：git commit -m "feat: add typography section"

<p class="ody-truncate">单行省略：很长的文本……</p>
<p class="ody-line-clamp-2">两行钳制：较长的说明文字……</p>
<p class="ody-line-clamp-3">三行钳制：更长的说明文字……</p>
<p class="ody-nowrap">保持单行不换行的内容</p>
